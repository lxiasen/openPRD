## 设计系统核心亮点

### 色彩体系
- **主色调**：深蓝 Slate 色系（#0F172A ~ #F8FAFC），传达专业稳重
- **品牌强调色**：青绿色 Cyan（#0891B2），传达智能高效
- **功能色**：红/黄/绿三色风险等级体系
- **语义化背景**：每个功能色都有对应的浅色背景用于标签、提示

### 字体系统
- **中文优先**：系统字体栈，确保跨平台一致性
- **等宽字体**：用于代码、ID显示
- **6级字号**：从Display(32px)到Overline(11px)，层次分明

### 组件规范
- **按钮**：4种类型（主/次/文字/危险），统一高度和圆角
- **标签**：6种语义色，统一尺寸和圆角
- **卡片**：统一阴影和圆角，营造层次感
- **表格**：粘性表头，行Hover效果，选中高亮

## 页面设计核心亮点

### 1. 工作台（首页）
- **数据概览卡片**：4张卡片展示核心指标，带环比数据
- **最近项目列表**：表格展示，状态标签彩色区分
- **快捷操作**：主按钮突出「新建PRD质检项目」

### 2. 新建项目弹窗
- **双模式导入**：粘贴Markdown / 上传文件
- **快捷操作**：「创建并一键质检」按钮，减少操作步骤
- **实时验证**：表单验证和错误提示

### 3. PRD原文编辑器Tab
- **左右分栏**：左侧大纲导航（20%）+ 右侧编辑器（80%）
- **模糊点联动**：大纲显示模糊点数量，点击跳转
- **双模式切换**：编辑模式（Markdown）/ 预览模式（渲染）
- **版本管理**：底部版本栏，支持历史版本回溯

### 4. 质检报告Tab
- **统计筛选栏**：5张统计卡片 + 多维度筛选
- **检查结果表格**：可展开行，显示完整检查项信息
- **风险分级**：高/中/低风险彩色标签
- **批量操作**：勾选多行，批量标记/导出

### 5. 优化版PRD Tab
- **AI生成标识**：紫色主题，区分AI生成内容
- **修改高亮**：蓝色背景和边框标记AI修改
- **撤销功能**：支持单条撤销修改
- **修改联动**：Hover显示修改说明和检查项详情

### 6. 差异对比与修改确认Tab
- **并排对比**：左侧原文（红色删除）+ 右侧优化版（绿色新增）
- **差异指示器**：中间分割区显示修改状态和连接线
- **右侧联动面板**：点击修改行显示详情，支持接纳/拒绝
- **批量操作**：全选接纳/拒绝，快速处理

### 7. 导出与归档弹窗
- **成功提示**：绿色渐变背景，成功图标动画
- **双Tab预览**：最终PRD预览 / 质检报告预览
- **多格式导出**：支持 Markdown / Word / PDF / Excel
- **归档功能**：项目归档管理

## 核心交互设计

### 1. 全链路ID绑定
- 检查项ID（CHECK-001）贯穿全流程
- 从检查项 → 修改点 → 差异对比 → 接纳/拒绝，全程可追溯

### 2. 强联动的上下文体验
- 点击检查项 → 定位PRD原文段落
- 点击修改行 → 查看检查项详情
- 点击大纲 → 编辑器滚动到对应位置

### 3. 风险分级机制
- 高/中/低风险三色标签
- 高风险项优先展示，强制提醒
- 统计卡片顶部彩色边框区分

### 4. 非破坏性修改原则
- 原始PRD永不覆盖
- 优化版独立存在
- 差异对比让用户100%决策
- 拒绝的修改完全回退原文

### 5. 实时反馈
- 自动保存提示
- 操作成功Toast
- 进度条显示
- 撤销操作支持

### 推荐技术栈
- **UI框架**：Tailwind CSS（原子化CSS，便于实现设计规范）
- **组件库**：Headless UI（无样式组件，配合Tailwind）
- **图标**：Lucide React（与设计稿一致）
- **Markdown编辑器**：@uiw/react-md-editor 或 react-markdown
- **Diff展示**：react-diff-viewer 或自研diff组件
- **状态管理**：Zustand 或 Redux Toolkit
- **动画**：Framer Motion（弹窗、页面切换动画）

## 设计稿使用说明

1. **设计系统规范**是所有页面的基础，定义了色彩、字体、间距等基础令牌
2. **每个页面设计稿**包含：
   - 页面概述和目标
   - 整体布局结构（ASCII图示）
   - 各区域的详细设计规范
   - 组件的具体样式参数
   - 交互细节和动画说明
   - 空状态和异常状态

3. **实现优先级建议**：
   - P0：设计系统 → 工作台 → 新建项目弹窗 → PRD原文Tab
   - P1：质检报告Tab → 优化版PRD Tab
   - P2：差异对比Tab → 导出弹窗

4. **设计稿中的数值**：
   - 所有颜色值为HEX格式
   - 所有尺寸单位为px
   - 所有间距基于4px网格系统
   - 阴影值可直接用于CSS box-shadow
