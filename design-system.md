# openPRD - 设计系统规范

## 一、设计理念

### 设计原则
1. **清晰专业**：界面层次分明，信息架构清晰，符合B端产品专业气质
2. **智能高效**：通过色彩、动效传达AI智能感，操作流程高效顺畅
3. **可信稳重**：棕色主色调传达可信赖感，适合企业级应用场景
4. **聚焦内容**：弱化UI装饰，让PRD内容成为视觉焦点

### 整体风格
- 现代简约风格，大量留白，呼吸感强
- 微圆角设计（4-8px），柔和专业
- 细腻的阴影层次，营造空间感
- 精致的图标系统，线性风格为主

## 九、动画与过渡

### 过渡时长
| 类型 | 时长 | 用途 |
|------|------|------|
| 快速 | 150ms | 按钮状态、小元素 |
| 标准 | 200ms | 卡片、下拉菜单 |
| 中等 | 300ms | 弹窗、侧边栏 |
| 缓慢 | 400ms | 页面切换 |

### 缓动函数
- 标准：`cubic-bezier(0.4, 0, 0.2, 1)`
- 进入：`cubic-bezier(0, 0, 0.2, 1)`
- 退出：`cubic-bezier(0.4, 0, 1, 1)`
- 弹性：`cubic-bezier(0.34, 1.56, 0.64, 1)`（仅用于小元素）
