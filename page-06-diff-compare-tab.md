# 页面6：项目详情页 - 差异对比与修改确认Tab 视觉设计稿

## 页面概述

**页面目标**：实现原文与优化版的可视化diff对比，支持单条/批量接纳/拒绝修改，完成最终PRD的确认

**布局结构**：顶部操作栏 + 左右分栏对比区 + 右侧检查项联动面板

**页面尺寸**：1440px × 900px

## 八、交互细节

### 修改行选择
- 点击Checkbox选中
- 点击行任意位置展开详情
- 选中后右侧面板显示批量操作

### 接纳/拒绝操作
- 点击后立即生效
- 视觉反馈：行背景闪烁
- 统计数字实时更新
- 显示Toast提示

### 撤销操作
- 操作后3秒内可撤销
- Toast显示撤销按钮
- 撤销后恢复原状态

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `↑/↓` | 上下切换修改行 |
| `Space` | 选中/取消选中 |
| `A` | 接纳当前修改 |
| `R` | 拒绝当前修改 |
| `Ctrl/Cmd + A` | 全选 |
| `Ctrl/Cmd + Enter` | 生成最终PRD |
