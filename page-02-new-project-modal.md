# 页面2：新建项目弹窗视觉设计稿

## 弹窗概述

**弹窗目标**：快速完成项目创建与PRD原文导入，降低用户启动门槛

**弹窗尺寸**：600px宽 × 560px高

**动画效果**：
- 进入：从上方滑入 + 淡入，300ms，ease-out
- 退出：向上滑出 + 淡出，200ms，ease-in
- 背景遮罩：淡入淡出，200ms

---

## 整体布局结构

```
┌─────────────────────────────────────────────────────────────┐
│  弹窗标题                                        [关闭按钮 X] │  ← 56px
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  表单区域                                                    │  ← 剩余高度
│  - 项目名称                                                  │
│  - 项目备注                                                  │
│  - PRD导入（Tab切换）                                         │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  底部操作栏                                      [取消] [创建] │  ← 64px
└─────────────────────────────────────────────────────────────┘
```
