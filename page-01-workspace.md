# 页面1：工作台（首页）视觉设计稿

## 页面概述

**页面目标**：用户进入产品的第一触点，快速发起新项目、查看历史项目、掌握整体质检数据

**布局结构**：上下分栏 + 左右卡片布局

**页面尺寸**：1440px × 900px（标准桌面端）

## 五、交互状态详解

### 表格行Hover效果
- 背景渐变：`#F8FAFC`
- 过渡：150ms ease
- 操作按钮显示：默认透明度0.6，Hover时1

### 状态标签动画
- 「质检中」状态的圆点有呼吸动画：
  ```css
  animation: pulse 2s cubic-bezier(0.4, 0, 0.6, 1) infinite;
  ```

### 按钮点击反馈
- 主按钮点击：scale(0.98)，100ms
- 次按钮点击：背景加深

### 卡片Hover效果
- 阴影升级：`0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- 轻微上移：translateY(-2px)
- 过渡：200ms ease

## 六、视觉层次与重点

### 视觉焦点顺序
1. **主按钮「新建PRD质检项目」**：品牌色+阴影，最吸睛
2. **数据卡片大数字**：高对比度，快速传达价值
3. **表格状态标签**：彩色标签，区分项目状态
4. **操作按钮**：Hover时显现，不干扰阅读

### 信息密度控制
- 大量使用留白（24px间距）
- 卡片阴影柔和，不抢眼
- 次要信息使用灰色系，降低视觉权重
