# 页面7：最终PRD导出与归档弹窗 视觉设计稿

## 弹窗概述

**触发路径**：差异对比页点击「生成最终PRD」后，弹出确认弹窗，进入导出页

**弹窗尺寸**：720px宽 × 600px高

**动画效果**：
- 进入：从下方滑入 + 淡入，300ms，ease-out
- 退出：向下滑出 + 淡出，200ms，ease-in

---

## 九、交互细节

### Tab切换
- 切换时内容区域淡入淡出
- 动画：150ms ease

### 预览滚动
- 最终PRD预览可滚动查看
- 滚动条样式统一

### 导出操作
- 点击导出后显示进度
- 完成后自动下载
- 下载失败显示错误提示

### 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `Esc` | 关闭弹窗 |
| `Tab` | 切换Tab |
| `Ctrl/Cmd + S` | 导出PRD |
| `Ctrl/Cmd + P` | 导出报告 |
