# 页面3：项目详情页 - PRD原文编辑器Tab 视觉设计稿

## 页面概述

**页面目标**：展示用户导入的原始PRD，支持在线编辑、版本回溯，与质检报告强联动

**布局结构**：顶部项目信息栏 + 左右分栏（左侧大纲20% + 右侧编辑器80%）

**页面尺寸**：1440px × 900px（标准桌面端，内容区高度自适应）

## 七、交互细节

### 大纲与编辑器联动

**点击大纲跳转**
- 点击大纲项，编辑器平滑滚动到对应段落
- 滚动动画：300ms ease
- 目标段落短暂高亮（背景`#ECFEFF`，2秒后恢复）

**编辑器滚动同步大纲**
- 编辑器滚动时，自动高亮当前可见的最高层级大纲项
- 高亮样式：选中状态

### 模糊点标记交互

**点击模糊点提示**
- 点击模糊点徽章或提示条
- 自动切换到「质检报告」Tab
- 自动筛选显示对应检查项
- 检查项行高亮显示

### 编辑模式交互

**自动保存**
- 停止输入2秒后自动保存
- 底部显示「已自动保存」提示
- 版本号不变（覆盖当前版本）

**快捷键**
- `Ctrl/Cmd + S`：手动保存
- `Ctrl/Cmd + B`：粗体
- `Ctrl/Cmd + I`：斜体
- `Ctrl/Cmd + K`：插入链接
